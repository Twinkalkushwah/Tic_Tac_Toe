## Installation

1. Clone the repository:
   ```sh
   git clone <your-repo-url>
   cd tic-tac-toe-react
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm start
   ```
